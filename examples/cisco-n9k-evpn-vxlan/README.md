# EVPN/VXLAN Fabric with Cisco Nexus 9000v

This example demonstrates deploying a leaf-spine EVPN/VXLAN fabric with vPC multi-homing using the Network Operator.

For detailed setup instructions and configuration explanations, see the [EVPN/VXLAN Fabric Tutorial](https://ironcore-dev.github.io/network-operator/tutorials/evpn-vxlan-fabric).
